📌 项目概述（gpt写的介绍，我不会写作文）
SentinelShield 是一个开源的、分布式、协同式DDoS防御节点系统，旨在为关键网络基础设施提供实时、智能、可扩展的DDoS攻击防护能力。不同于传统单一防火墙解决方案，SentinelShield采用集群协同防御架构，多个防御节点组成互联网络，共享威胁情报，实现全局攻击态势感知和快速响应。

本项目核心是一个轻量级但功能强大的Python节点程序，可部署在服务器边缘，作为第一道防线拦截恶意流量，同时保持正常业务流量畅通。通过智能流量分析、自适应防御策略和集群协同机制，SentinelShield能在攻击初期就有效抑制威胁，防止服务中断。

✨ 核心功能
🔍 智能攻击检测
多维度流量分析：实时监控包速率(PPS)、带宽(Mbps)、连接数等关键指标
异常行为识别：基于统计模型检测SYN Flood、UDP Flood、HTTP Flood等常见攻击
置信度评估：为每次攻击判断分配置信度分数，减少误报
🛡️ 自适应防御
三级防御模式：
normal：常规监控，仅记录可疑活动
alert：中等响应，对高置信度攻击自动拉黑
aggressive：强力响应，主动限制可疑流量，启用黑洞路由
动态阈值调整：根据业务流量基线自动调整检测阈值
白名单保护：重要IP永不被阻断（如监控系统、合作伙伴）
🌐 集群协同防御
威胁情报共享：节点间实时同步可疑IP和攻击特征
级联防御：一个节点检测到的攻击，全网自动防御
分布式黑名单：中央协调器同步黑名单，确保策略一致性
健康监控：节点心跳机制，自动检测离线节点
💻 交互式管理
TTY独占命令面板：提供直观的终端界面，无日志干扰
实时状态可视化：显示流量统计、攻击状态、EasyTier网络状态
日志隔离：所有日志写入文件，通过logs命令查看
配置热更新：无需重启即可调整防御参数
